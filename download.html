How to Add ARES to Your Server

Follow the steps below to get ARES working on your server:

  1. Click the Add to Server button above to authorize ARES for your server.
  2. Once authorized, you can start tracking stats for your Discord server instantly!
  3. If you want to view commands or see source code, visit the GitHub repository.
